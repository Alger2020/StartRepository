# 类的定义  

## c++语法：定义类  
``` c++
 class 类名 //类声明(Declaration)部分  
{
public:  
	公有成员  
protected:  
	保护成员  
private:  
	私有成员  
}；  

 各函数成员的完整定义 //类实现(Implementation)部分  
```  
 ******  
### 语法说明  
* **class**是定义类时的关键字；  
* **类名**需要符号标识符的命名规则；  
* **类成员**有两种，分别是数据成员和函数成员。某些特殊的 类可能只包含1种成员，比如只包含数据成员，或只包含函数成员；
